##说明

此项目为简易自定义组件拖拽

其中包含：

组件全局引入 边界极限

组件拖拽

自定义宽高（未写成独立的方法）

四分布局 六分布局切换 （可自定义）

设置布局后拖拽组件可直解放置对应位置

清除布局

清除组件功能

（后续会逐步添加其他方法）
